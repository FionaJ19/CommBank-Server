Project Overview

CommBank-Server is a backend service built with ASP.NET Core Web API and MongoDB. It manages banking goals for users, including savings targets, balances, and target dates.

Main features:

Create, read, update, and delete user goals (CRUD)

Retrieve goals by user

Connects and interacts with MongoDB database

Provides JSON API for frontend integration

Tech Stack

Backend Framework: ASP.NET Core 7.0

Database: MongoDB

Language: C#

Dependencies:

MongoDB.Driver for MongoDB operations

Microsoft.AspNetCore.Mvc for REST API support
